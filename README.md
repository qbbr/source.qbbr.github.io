qbbr.io blog source
===================

installation
------------

```bash
sudo apt install ruby-full
sudo gem install bundler jekyll
sudo gem install rdoc jekyll jekyll-tagging jekyll-less therubyracer
```

run
---

```bash
cd blog-source/
jekyll serve -w

bundle exec jekyll serve
```

open in browser [blog](http://127.0.0.1:4000/)
